## 🤖 Automation & Web Scraping

Fokus utama adalah membangun sistem otomatisasi tangguh yang memiliki ketahanan tinggi terhadap *detection mechanisms* (Cloudflare, CAPTCHA, Rate-limiting).

* **Headless Browser Automation**:
  * **Puppeteer & Playwright**: Mengotomatisasi alur kerja web kompleks, simulasi klik pengguna, pengisian formulir, dan bypass login.
  * **Selenium**: Digunakan untuk pengujian otomatis cross-browser dan integrasi enterprise.
* **Scraping Frameworks**:
  * **Scrapy**: Web scraping terdistribusi berkinerja tinggi dengan pipeline terstruktur.
  * **BeautifulSoup4 & HTTPX / Requests**: Pengambilan data cepat melalui parsing HTML dan HTTP APIs.
* **Bypass & Evasion Techniques**:
  * Manajemen proxy dinamis (Proxy Rotation & Pool Management).
  * Pemecahan CAPTCHA pihak ketiga (2Captcha, Anti-Captcha).
  * Penyamaran fingerprint browser (stealth plugins).

## 🌐 Backend & API Development

Membangun REST API dan sistem backend untuk mendukung arsitektur otomatisasi dan integrasi frontend.

* **Runtime & Frameworks**:
  * **Node.js**: Arsitektur event-driven, penanganan operasi asynchronous I/O tinggi.
  * **Express.js**: Pembuatan server API ringan dan proxy keamanan.
  * **FastAPI & Flask**: Pembuatan endpoint Python berkecepatan tinggi dengan dokumentasi otomatis (Swagger).
* **Database & Caching**:
  * **PostgreSQL / MySQL**: Database relasional untuk transaksi data sensitif.
  * **MongoDB**: Database NoSQL untuk penyimpanan data hasil scraping tidak terstruktur.
  * **Redis**: Caching berkecepatan tinggi, broker antrian pesan (Queue Management).
