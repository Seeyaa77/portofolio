# 🌌 Muhammad Raffli Aldiansyah — Personal Portfolio

A sleek, premium, and interactive personal portfolio website designed with a **Monochromatic Cyberpunk-Lite** aesthetic. Built to highlight expertise in **Automation Engineering, Web Scraping, API Orchestration, and Web Development**.

## 🎨 Design Philosophy & Features

- **Monochromatic Cyberpunk-Lite Theme**: Deep charcoal backgrounds (`#0a0a0a`), clean glassmorphic components, subtle grid overlays, and elegant neon cyan-blue accents.
- **Glassmorphic CTA Connect Button**: A highly polished, interactive "Connect with me" button with a custom hover glow and fluid vector transition.
- **Smooth Metrics Counters**: Scroll-triggered counting animation for key stats (e.g., 80+ Projects Delivered, 24/7 Automation Uptime) with a memory-safe custom trigger hooks.
- **Clean Academic Timeline**: A beautiful, minimalist vertical education timeline featuring custom hover cards, responsive structure, and custom badges indicating current status.
- **Fully Responsive & Smooth Animations**: Fluid responsive layout from desktop down to mobile viewports, powered by Framer Motion.

## 🛠️ Technology Stack

- **Framework**: [React](https://react.dev/) + [Vite](https://vite.dev/) (powered by the high-performance `rolldown-vite` bundler)
- **Styling**: Tailwind CSS v4 & custom Vanilla CSS for advanced animations and glassmorphism.
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### 📋 Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (recommended version `18.x` or higher).

### 🔧 Installation

Clone this repository and install dependencies using your package manager of choice:

```bash
# Using npm
npm install

# Using pnpm
pnpm install
```

### 💻 Local Development

Start the Vite development server locally:

```bash
# Using npm
npm run dev

# Using pnpm
pnpm run dev
```

The application will be running at `http://localhost:5173` (or the next available port).

### 📦 Production Build

Compile the application to a highly optimized bundle:

```bash
# Using npm
npm run build

# Using pnpm
pnpm run build
```

The production-ready assets will be generated in the `dist/` directory.

## ☁️ Deployment

This project is configured for seamless deployment to **Vercel** or any static hosting platform.

### Deploying to Vercel

```bash
# Install Vercel CLI if you haven't already
npm install -g vercel

# Deploy to production
vercel --prod
```

> [!NOTE]
> If Vercel has issues installing dependencies due to lockfile conflicts, ensure you specify the correct package manager override in the Vercel project settings.
